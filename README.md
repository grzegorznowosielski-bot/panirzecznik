# panirzecznik.pl

Landing VSL dla [panirzecznik.pl](https://panirzecznik.pl) — statyczna strona (HTML/CSS/JS) z formularzem PHP.

## Struktura

- `index.html` — strona główna
- `styles.css`, `script.js` — front
- `submit.php` — wysyłka formularza (dhosting)
- `video/hero.mp4` — wideo w sekcji hero

## Wdrożenie (FTP)

Wgraj zawartość tego folderu do katalogu domeny na dhosting (`public_html` lub katalog przypisany do domeny). Zachowaj strukturę plików.

Po wgraniu ustaw uprawnienia `submit.php` na `644` (lub według wymagań hostingu).

## Lokalnie

Otwórz `index.html` w przeglądarce lub użyj prostego serwera HTTP. Formularz wymaga PHP na serwerze.
